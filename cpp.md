# C++

## Imprimir na tela

```cpp
#include <iostream>

int main() {

  std::cout << "Hello, World!" << std::endl;
  
  return 0;
}
```

## Capturar dados do teclado

```cpp
#include <iostream>

int main() {

  char * text;
  std::cout << "Type something: ";
  std::cin >> text;
  
  return 0;
}
```
